# Readme

This code is published to accompany the paper "Changing lanes? Exploring mergers and acquisitions in the global auto sector through GPN 2.0".

Unfortunately, due to restrictions on the proprietary M&A data we used for this analysis - provided by BvD - we are not able to share a copy of that underlying dataset itself.

However, we are able to share the code that we used to produce the results that are published in our paper. This code files are numerically ordered, and are intend to be run one after the other to produce the final figures and tables found in the paper.
